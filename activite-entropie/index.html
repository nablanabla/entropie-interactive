Le jeu des gobelets

🎯 Mise en situation

Le jeu : Une balle est cachée sous l'un de plusieurs gobelets. Vous devez la trouver en posant des questions à un informateur qui connaît l'emplacement de la balle mais qui ne sait répondre que par OUI ou par NON.

Votre objectif est de minimiser le nombre de questions nécessaires.

L'idée est la suivante : plus le nombre de questions nécessaires est grand, plus l'information a de la valeur.

Analogie scientifique : trouver la bonne hypothèse en minimisant le nombre d'expérience.

Analogie médicale : trouver le bon diagnostic en minimisant le nombre de test.

Quelle formule relie le nombre de questions à la probabilité p ?

A Nombre de questions = p
B Nombre de questions = $1/p$
C Nombre de questions = $\log_₂(p)$
D Nombre de questions = $- \log_2(p)$

Hint : La fonction mathématique $\log_2(x)$ est la fonction qui permet d'extraire la puissance lorsqu'un nombre x est écrit en base 2.

Exemple: $\log_2(2^1)=1, \log_2(2^2)=2, \log_2(2^3)=3, \ldots, \log_2(2^n)=n$

🔑 Formule de l'entropie de Shannon (cas équiprobable):

$H=-\log_2(p)$

Dans le cas d'événements équiprobables, l'information la balle est sous un des gobelets « vaut » $-\log_2(p)$.

📊 Visualisation de la fonction

💡 Interprétation :

Plus la probabilité p est petite, plus l'information nécessaire $-\log_2(p)$ est grande.

Autrement dit : un événement rare (p faible) apporte beaucoup d'information quand il se produit, tandis qu'un événement très probable (p proche de 1) apporte peu d'information.

A 1 question

🎓 L'Entropie de Shannon

L'entropie mesure le nombre moyen de questions (en bits) nécessaires pour identifier un événement parmi plusieurs possibilités.

Formule générale de l'entropie :

$$H=-\displaystyle \sum_i p_i \times \log_2(p_i)$$

où $p_i$ est la probabilité de chaque événement indicé sur $i$

🔍 Cas particulier :

Si tous les $p_i$ sont égaux ($p_i=\frac{1}{n}$), on retrouve :

$$H=\log_2(n)=-\log_2(p)$$

C'est la formule que nous avons découverte avec les gobelets équiprobables !

Que signifie une entropie élevée ?

A Les événements sont très prévisibles
B Il y a beaucoup d'incertitude, les événements sont équiprobables
C Un seul événement est possible
D Le système est en équilibre

🤖 Connexion avec l'IA

Cette formalisation de l'information devient le socle des modèles de langage actuels comme GPT !

Les modèles sont entraînés à minimiser l'entropie croisée, c'est-à-dire à maximiser la probabilité des mots qui apparaissent dans les textes.

La perplexité, métrique standard pour évaluer un modèle de langage, est directement dérivée de l'entropie : Perplexité = 2^H(X)
